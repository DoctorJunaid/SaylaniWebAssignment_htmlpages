Introduction to Computer Science

Explore the fundamentals of computer science, programming, and algorithms in this introductory course.

Course Description

This course provides an introduction to the basic concepts of computer science. Students will learn about algorithms, data structures, programming languages, and how to approach problem-solving using computers. The course is designed for beginners with no prior programming experience.

Topics Covered

  • Introduction to Computers and Programming
  • Data Types, Variables, and Operators
  • Control Structures (Loops, Conditionals)
  • Functions and Recursion
  • Arrays and Data Structures
  • Algorithms and Sorting Techniques
  • Introduction to Object-Oriented Programming
  • Basic Debugging and Testing Techniques

Instructor Details

Dr. Ali Raza

Dr. Ali Raza is an experienced computer scientist with a passion for teaching. She holds a PhD in Computer Science from MIT and has been teaching introductory courses for over 10 years. Her research interests include artificial intelligence and machine learning.
